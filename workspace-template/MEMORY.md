# MEMORY.md — Curated Long-Term Memory

Adam's assistant's curated long-term memory. Raw daily notes live in `memory/YYYY-MM-DD.md`; the distilled, still-load-bearing truths live here.

## Scope

- Significant decisions, ongoing initiatives, and context that outlives any single day.
- Preferences confirmed over time (tone, phrasing, tool choices).
- Outcomes and lessons — what worked, what didn't, and why.

## Rules

- **Main sessions only.** Do not load this file in shared / group / third-party contexts.
- Edit freely in main sessions; prune when a memory stops mattering.
- Prefer short, dated entries over long prose.
- If a memory is about firm operating rules, update `JSP-CONTEXT.md` instead and cross-reference here.
